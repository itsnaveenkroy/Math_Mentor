# Probability: Basic Concepts

## Fundamental Definitions
- Sample Space (S): Set of all possible outcomes
- Event (E): Subset of sample space
- P(E) = Number of favorable outcomes / Total outcomes (equally likely)

## Axioms of Probability
- 0 ≤ P(E) ≤ 1
- P(S) = 1
- P(∅) = 0
- P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

## Complementary Events
P(A') = 1 - P(A)

## Conditional Probability
P(A|B) = P(A ∩ B) / P(B), where P(B) > 0

## Independent Events
A and B are independent if:
P(A ∩ B) = P(A) × P(B)
Equivalently: P(A|B) = P(A)

## Mutually Exclusive Events
A and B are mutually exclusive if:
P(A ∩ B) = 0
Then: P(A ∪ B) = P(A) + P(B)

## Common Mistakes
- Confusing independent and mutually exclusive events
- Forgetting to subtract intersection in union formula
- Assuming equally likely outcomes when they aren't
