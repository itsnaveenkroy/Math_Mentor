# Algebra: Systems of Equations

## Methods for Solving
1. **Substitution**: Express one variable in terms of another
2. **Elimination**: Add/subtract equations to eliminate a variable
3. **Matrix method**: Use Cramer's rule or row reduction

## Two Variables
ax + by = c
dx + ey = f
- Unique solution if ae - bd ≠ 0
- No solution if ae - bd = 0 and cf ≠ bf (inconsistent)
- Infinite solutions if ae - bd = 0 and equations are proportional

## Three Variables
Use elimination or Cramer's rule with 3×3 determinants.

## Homogeneous Systems
ax + by + cz = 0
dx + ey + fz = 0
gx + hy + iz = 0
- Always has trivial solution (0,0,0)
- Non-trivial solution exists if det = 0

## Word Problem Strategy
1. Define variables clearly
2. Translate conditions into equations
3. Count: need as many independent equations as unknowns
4. Solve using appropriate method
5. Verify solution satisfies all original conditions

## Common Mistakes
- Losing a solution by dividing by a variable (could be 0)
- Not checking if the system is consistent
- Writing wrong equations from word problems
