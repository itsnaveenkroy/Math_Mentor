# Domain and Range Constraints

## Domain Rules
1. **Denominator ≠ 0**: For f(x) = g(x)/h(x), require h(x) ≠ 0
2. **Even roots ≥ 0**: For √(f(x)), require f(x) ≥ 0
3. **Logarithm argument > 0**: For log(f(x)), require f(x) > 0
4. **Logarithm base**: Must be positive and ≠ 1
5. **Inverse trig domains**:
   - sin⁻¹(x): -1 ≤ x ≤ 1
   - cos⁻¹(x): -1 ≤ x ≤ 1
   - tan⁻¹(x): all real numbers

## Range Rules
1. **Trigonometric**: -1 ≤ sin(x), cos(x) ≤ 1
2. **Quadratic**: y = ax² + bx + c has range [-D/4a, ∞) if a > 0, (-∞, -D/4a] if a < 0
3. **Exponential**: e^x > 0 always
4. **Absolute value**: |f(x)| ≥ 0 always

## Checking Answers Against Constraints
After solving, always verify:
- Solution is in the domain of the original problem
- Answer is physically meaningful (e.g., lengths > 0, probabilities 0-1)
- Units are consistent

## Common Domain Pitfalls
- Getting log(-5) as an answer → extraneous solution
- Computing √(negative) in real number problems
- Division by zero at a particular x value
- Probability > 1 or < 0 → error in calculation
