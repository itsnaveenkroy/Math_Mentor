# Calculus: Optimization (Maxima and Minima)

## First Derivative Test
1. Find f'(x) = 0 (critical points)
2. Check sign change of f'(x):
   - f' changes + to -: Local maximum
   - f' changes - to +: Local minimum
   - No sign change: Neither (inflection point)

## Second Derivative Test
At critical point x = c where f'(c) = 0:
- f''(c) > 0: Local minimum
- f''(c) < 0: Local maximum
- f''(c) = 0: Test is inconclusive, use first derivative test

## Global (Absolute) Extrema on [a, b]
1. Find all critical points in (a, b)
2. Evaluate f at critical points and endpoints
3. Largest value = absolute maximum
4. Smallest value = absolute minimum

## Optimization Problem Strategy (JEE)
1. Read problem and identify what to maximize/minimize
2. Express the objective function in ONE variable
3. Find domain/constraints
4. Apply derivative tests
5. Verify answer makes physical sense

## Common Optimization Setups
- Maximizing area with perimeter constraint
- Minimizing material for given volume
- Finding shortest distance
- Maximizing profit/revenue

## AM-GM for Optimization
For positive numbers: min of (x + a/x) = 2√a, achieved when x = √a

## Common Mistakes
- Not checking endpoints of the domain
- Finding extrema but not verifying they're max vs min
- Forgetting physical constraints (lengths > 0, etc.)
- Not expressing in single variable before differentiating
