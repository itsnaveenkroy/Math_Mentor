# Algebra: Quadratic Equations

## Quadratic Formula
For ax² + bx + c = 0, the solutions are:
x = (-b ± √(b² - 4ac)) / (2a)

## Discriminant
D = b² - 4ac
- D > 0: Two distinct real roots
- D = 0: One repeated real root (equal roots)
- D < 0: Two complex conjugate roots

## Vieta's Formulas
For roots α and β of ax² + bx + c = 0:
- Sum of roots: α + β = -b/a
- Product of roots: α × β = c/a

## Nature of Roots
- If a > 0: parabola opens upward, minimum at vertex
- If a < 0: parabola opens downward, maximum at vertex
- Vertex: x = -b/(2a), y = -D/(4a)

## Common Mistakes
- Forgetting ± in the quadratic formula
- Sign errors when applying Vieta's formulas
- Not checking if a = 0 (which makes it linear, not quadratic)
