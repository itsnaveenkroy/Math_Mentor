# Algebra: Inequalities and Absolute Values

## Properties of Inequalities
- Adding/subtracting same number preserves direction
- Multiplying/dividing by positive number preserves direction
- Multiplying/dividing by negative number REVERSES direction

## Absolute Value
- |x| = x if x ≥ 0, -x if x < 0
- |x| ≥ 0 always
- |xy| = |x|·|y|
- |x/y| = |x|/|y| (y ≠ 0)
- |x + y| ≤ |x| + |y|  [Triangle Inequality]
- ||x| - |y|| ≤ |x - y|

## Solving |f(x)| < a (where a > 0)
Equivalent to: -a < f(x) < a

## Solving |f(x)| > a (where a > 0)
Equivalent to: f(x) < -a OR f(x) > a

## Quadratic Inequalities
For ax² + bx + c > 0:
1. Find roots using quadratic formula
2. Determine sign in each interval
3. Use sign chart / wavy curve method

## Wavy Curve Method (JEE Important)
For f(x) = (x-a₁)^n₁(x-a₂)^n₂... :
1. Find all roots and arrange on number line
2. Start from rightmost point (positive)
3. At each root: change sign if odd multiplicity, same sign if even

## Common Mistakes
- Forgetting to flip inequality when multiplying by negative
- Squaring both sides without checking sign
- Not checking domain restrictions
