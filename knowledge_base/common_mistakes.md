# Common Mistakes and Pitfalls in JEE Mathematics

## Algebra Mistakes
1. **Sign errors**: Most common mistake in algebra. Double-check every sign change.
2. **Domain issues**: Always check domain of logarithms (x > 0), even denominators (≠ 0), square roots (≥ 0).
3. **Extraneous solutions**: When squaring both sides, verify solutions in original equation.
4. **Modular arithmetic errors**: |x| = x only when x ≥ 0.

## Probability Mistakes
1. **Not using complement**: For "at least one" problems, always consider P(≥1) = 1 - P(0).
2. **Overcounting**: Use inclusion-exclusion carefully.
3. **Independence assumption**: Don't assume events are independent unless stated.
4. **Confusing nPr and nCr**: Ask "does order matter?"

## Calculus Mistakes
1. **Chain rule forgotten**: Most differentiation errors come from missing the chain rule.
2. **Integration constant**: Always add +C for indefinite integrals.
3. **L'Hôpital misuse**: Only apply for 0/0 or ∞/∞ forms.
4. **Domain of derivative**: Derivative may not exist at corners, cusps, or discontinuities.
5. **Optimization without constraints**: Always identify valid domain.

## Linear Algebra Mistakes
1. **Matrix multiplication order**: AB ≠ BA in general.
2. **Singular matrix inverse**: Check det ≠ 0 before finding inverse.
3. **Dimension mismatch**: Verify dimensions before operations.

## General Strategy for JEE
1. Read the problem twice
2. Identify the topic and sub-topic
3. Write down given information and what's asked
4. Choose the method (formula vs. first principles)
5. Solve step by step
6. Verify using alternative method or substitution
7. Check boundary/edge cases
8. Verify units and dimensions make sense
