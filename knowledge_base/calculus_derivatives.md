# Calculus: Derivatives

## Definition
f'(x) = lim(h→0) [f(x+h) - f(x)] / h

## Basic Derivative Rules
- d/dx (c) = 0
- d/dx (x^n) = nx^(n-1)  [Power Rule]
- d/dx (e^x) = e^x
- d/dx (a^x) = a^x × ln(a)
- d/dx (ln x) = 1/x
- d/dx (log_a x) = 1/(x × ln a)

## Trigonometric Derivatives
- d/dx (sin x) = cos x
- d/dx (cos x) = -sin x
- d/dx (tan x) = sec²x
- d/dx (cot x) = -csc²x
- d/dx (sec x) = sec x × tan x
- d/dx (csc x) = -csc x × cot x

## Inverse Trigonometric Derivatives
- d/dx (sin⁻¹x) = 1/√(1-x²)
- d/dx (cos⁻¹x) = -1/√(1-x²)
- d/dx (tan⁻¹x) = 1/(1+x²)

## Rules
- Product Rule: (fg)' = f'g + fg'
- Quotient Rule: (f/g)' = (f'g - fg') / g²
- Chain Rule: d/dx f(g(x)) = f'(g(x)) × g'(x)

## Applications (JEE)
- Finding maxima/minima: f'(x) = 0, check f''(x)
- Rate of change problems
- Tangent and normal to curves
- Rolle's Theorem: If f(a) = f(b), continuous on [a,b], differentiable on (a,b), then ∃c: f'(c) = 0
- Mean Value Theorem: ∃c in (a,b): f'(c) = (f(b)-f(a))/(b-a)

## Common Mistakes
- Forgetting chain rule in composite functions
- Sign errors in quotient rule
- Not checking second derivative for nature of critical points
