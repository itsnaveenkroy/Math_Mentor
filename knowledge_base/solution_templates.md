# Solution Templates

## Template: Quadratic Equation
1. Identify coefficients a, b, c in ax² + bx + c = 0
2. Calculate discriminant D = b² - 4ac
3. Apply quadratic formula: x = (-b ± √D) / 2a
4. Simplify the roots
5. Verify using Vieta's: sum = -b/a, product = c/a

## Template: Probability (Classical)
1. Define the sample space
2. Count total outcomes
3. Identify favorable outcomes
4. Calculate P = favorable / total
5. Simplify and verify 0 ≤ P ≤ 1

## Template: Conditional Probability
1. Identify events A and B
2. Find P(A ∩ B) using given information
3. Find P(B) (the condition)
4. Apply P(A|B) = P(A ∩ B) / P(B)
5. Verify with tree diagram if complex

## Template: Maxima/Minima
1. Express objective function f(x) in one variable
2. Find domain and constraints
3. Compute f'(x) and find critical points (f'(x) = 0)
4. Apply second derivative test or first derivative test
5. Check boundary values if domain is bounded
6. State the maximum/minimum value

## Template: Limit Evaluation
1. Try direct substitution first
2. If indeterminate (0/0, ∞/∞), try:
   a. Algebraic simplification (factoring, rationalizing)
   b. Standard limits
   c. L'Hôpital's Rule
   d. Taylor series expansion
3. For 1^∞ form: use e^(lim g(x)(f(x)-1))
4. Verify with numerical approximation

## Template: System of Linear Equations
1. Write in matrix form AX = B
2. Compute det(A)
3. If det(A) ≠ 0: unique solution using Cramer's rule or inverse
4. If det(A) = 0: check consistency
5. Verify solution by substituting back

## Template: Differentiation Application
1. Identify what needs to be found
2. Express in terms of a single variable
3. Differentiate using appropriate rules
4. Set derivative = 0 for critical points
5. Interpret the result in context
